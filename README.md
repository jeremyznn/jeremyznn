## À propos

Moi c’est **Jérémy**, aussi connu sous le pseudo **ZENOX**.

Je développe surtout des interfaces web avec **React**, **Next.js** et **Tailwind CSS**.
J’aime quand un projet est propre, clair, fluide et agréable à utiliser. Pas juste un truc joli en surface, mais quelque chose qui reste bien organisé dans le code.

Je touche aussi au back-end avec **Node.js**, **MongoDB** et **MySQL**, principalement pour créer des APIs, gérer des données et connecter les interfaces à de vraies fonctionnalités.

Je continue de progresser projet après projet, en essayant de garder une base simple : faire propre, utile, responsive et compréhensible.
